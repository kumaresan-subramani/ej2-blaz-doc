# Migration from Essential JS 1

This article describes the API migration process of MaskEdit component from Essential JS 1 to Essential JS 2.

## Common

| Behavior | API in Essential JS 1 | API in Essential JS 2 |
| --- | --- | --- |
| Adding custom class | **Property:** *cssClass* <br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "9999",<br/>cssClass: "custom"<br/>}); | **Property:** *cssClass*<br/><br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "9999",<br/>cssClass: "custom"<br/>});<br/>mask.appendTo("#mask"); |
| Destroy editor | Not Applicable | **Property:** *destroy*<br/><br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "00-000"<br/>});<br/>mask.appendTo("#mask");<br/>mask.destroy(); |
| Disable the maskedit control | **Method:** *disable* <br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "0000",<br/>value: “1234”<br/>});<br/>var maskObj = $(“#mask”).data(“ejMaskEdit”);<br/>maskObj.disable(); | **Can be achieved using**<br/>*API*: *enabled*<br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "99,999"<br/>});<br/>mask.appendTo("#mask");<br/>var maskObj = document.getElementById("mask").ej2\_instances[0];<br/>maskObj.enabled= false; |
| Enable the maskedit control | **Method:** *enable*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "0000",<br/>value: “1234”<br/>});<br/>var maskObj = $(“#mask”).data(“ejMaskEdit”);<br/>maskObj.enable(); | **Can be achieved using**<br/>*API*: *enabled*<br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "99,999"<br/>});<br/>mask.appendTo("#mask");<br/>var maskObj = document.getElementById("mask").ej2\_instances[0];<br/>maskObj.enabled= true; |
| Control state | **Property:** *enabled*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "00-000",<br/>enabled: false<br/>}); | **Property:** *enabled*<br/><br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "00-000",<br/>enabled: false<br/>});<br/>mask.appendTo("#mask"); |
| Persistence | **Property:** *enablePersistence*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "0000",<br/>enablePersistence: true<br/>}); | **Property:** *enablePersistence*<br/><br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "0000",<br/>enablePersistence: true<br/>});<br/>mask.appendTo("#mask"); |
| Triggers when editor is focused in | **Event:** *focusIn*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "00-00",<br/>focusIn: function() {}<br/>}); | **Event:** *focus*<br/><br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "0000",<br/>focus: function() {}<br/>});<br/>mask.appendTo("#mask"); |
| Triggers when editor is focused out | **Event:** *focusOut*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "0000",<br/>focusOut: function() {}<br/>}); | **Event** *blur* <br/><br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "00000",<br/>blur: function() {}<br/>});<br/>mask.appendTo('#mask"); |
| Sets height | **Property:** *height*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "0000",<br/>height : "30px"<br/>}); | **Can be achieved using,** <br/><br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "0000",<br/>cssClass: "custom"<br/>});<br/>mask.appendTo("#mask");<br/>**Css**<br/>.e-maskedtextbox.custom{<br/>height: 30px;<br/>} |
| HTML Attributes | **Property:** *htmlAttributes*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "0000",<br/>htmlAttributes: {name: "maskedtextbox"}<br/>}); | **Can be achieved using**<br/>**HTML**<br/>&lt;input id="mask" type="text" name="maskedtextbox" /><br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "0000"<br/>});<br/>mask.appendTo("#mask"); |
| Specifies Input mode | **Property:** *inputMode*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "0000",<br/>inputMode: ej.InputMode.Password<br/>}); | **Can be achieved using**<br/>**HTML**<br/>&lt;input id="mask" type="password" /><br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "0000",<br/>});<br/>mask.appendTo("#mask"); |
| Triggers on key press | **Event:** *keyPress*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "000",<br/>keyPress: function() {}<br/>}); | **Can be achieved using native event** <br/>**HTML**<br/> &lt;input id="mask" type="text" onkeypress="keyPress()" /><br/> &lt;script><br/>function keyPress(){}<br/>&lt;/script><br/> |
| Triggers on key up | **Event:** *keyUp*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "9999",<br/>keyUp: function() {}<br/>}); |**Can be achieved using native event** <br/>**HTML**<br/> &lt;input id="mask" type="text" onkeyup="keyUp()" /><br/> &lt;script><br/>function keyUp(){}<br/>&lt;/script><br/>  |
| Triggers on mouse out in maskedit control | **Event:** *mouseOut*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "0000-000",<br/>mouseOut: function() {}<br/>}); | **Can be achieved using native event** <br/>**HTML**<br/> &lt;input id="mask" type="text" onmouseout="mouseOut()" /><br/> &lt;script><br/>function mouseOut(){}<br/>&lt;/script><br/> |
| Triggers when mouse over in maskedit control | **Event:** *mouseOver*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "00-00",<br/>mouseOver: function() {}<br/>}); | **Can be achieved using native event** <br/>**HTML**<br/> &lt;input id="mask" type="text" onmouseover="mouseOver()" /><br/> &lt;script><br/>function mouseOver(){}<br/>&lt;/script><br/> |
| Name of maskedit control | **Property:** *name*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "0000",<br/>name: "pin"<br/>}); | **Can be achieved using**<br/>**HTML**<br/>&lt;input id="mask" type="text" name="maskedtextbox" /><br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "0000",<br/>});<br/>mask.appendTo("#mask"); |
| Triggers on keydown | **Event:** *onKeyDown*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "9999-9999",<br/>onKeyDown: function() {}<br/>}); | **Can be achieved using native event** <br/>**HTML**<br/> &lt;input id="mask" type="text" onkeydown="keyDown()" /><br/>&lt;script><br/>function keyDown(){}<br/>&lt;/script><br/> |
| Read only | **Property:** *readOnly*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "99-999",<br/>readOnly: true<br/>}); |**Can be achieved using,** <br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "0000",<br/>enabled: false, <br/>});<br/>mask.appendTo("#mask");<br/>**CSS**<br/>.e-mask{<br/> background-image: none !important;<br/>border-bottom-color:  rgba(0, 0, 0, 0.42) !important; <br/>} |
| Right to left | **Property:** *textAlign*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "9999",<br/>textAlign: "right"<br/>}); | **Property:** *enableRtl*<br/><br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "9999",<br/>enableRtl: true<br/>});<br/>mask.appendTo("#mask"); |
| Sets width | **Property:** *width*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "0000",<br/>width: "100px"<br/>}); | **Property:** *width*<br/><br/>var mask = new ej.inputs.MaskedTextBox({<br/>mask: "0000",<br/>width: "100px"<br/>});<br/>mask.appendTo("#mask"); |

| --- | --- | --- |

| Validation Rules | **Property:** *validationRules*<br/><br/>$("#mask").ejMaskEdit({<br/>maskFormat: "00-00",<br/>validationRules: {required: true}<br/>}); | **Validation in MaskedTextBox can be achieved through form validation**<br/>var options = {<br/> rules: {<br/>'maskValue': { required: [ true] },<br/> },<br/>}<br/>var formObject =  new ej.inputs.FormValidator('#form-element', options);<br/> formObject.customPlacement =  function(element, error) {<br/> document.querySelector(".form-group").appendChild(error);<br/>};<br/>var mask = new ej.inputs.MaskedTextBox({<br/>   mask: '000-000-0000',<br/>});<br/>mask.appendTo("#mask");<br/>**HTML** <br/> &lt;form id="form-element" class="form-horizontal"><br/>&lt;div class = "form-group"><br/>&lt;input id="mask" type="text" name="maskValue" class="form-control" /> <br/> &lt;div id="error">&lt;/div><br/>&lt;/div><br/>&lt;/form> |
