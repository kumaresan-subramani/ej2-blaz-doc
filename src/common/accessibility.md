# Accessibility

The Syncfusion JavaScript UI controls follows WAI-ARIA accessibility standard by default. This enables you to build accessible web application which are fully navigable by user with disabilities adn demo testing.

## Keyboard navigation

Keyboard navigation support enables users to interact with controls using keyboard shortcuts. Each control has its own set of shortcuts, refer the following documents to get the details of each control.
