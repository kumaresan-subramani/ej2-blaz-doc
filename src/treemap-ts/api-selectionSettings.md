# SelectionSettings

## Properties

### border [`BorderModel`](./api-borderModel.html)

To specifies the border

### enable `boolean`

To enable or disable the selection

Defaults to *false*

### fill `string`

To specifies the selection color

Defaults to *'#808080'*

### mode `string`

To specifies the selection mode.

Defaults to *'Item'*

### opacity `string`

To specified the opacity of color.

Defaults to *'0.5'*
